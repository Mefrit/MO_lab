Для выбора режима работы калькулятора, можно использовать следующие кнопки.

Так же имеется возможость возвращения предыдущих данных вычисления.

При пошаговом режиме самый оптимальный опорный элемент будет отображаться зеленым.

Остальные возможные элементы будут отображаться золотистым цветом.

Имеется возможность самостоятельно задать базис ( желательно использовать автоматический способ, тк возможны комбинации базисных переменных приводящие к неверному ответу).

Тут можно указать, те базисные переменные, которые вы желаете видеть базисны.ми
